# RS Liveries
DCS liveries for the RED STAR SQUADRON by `RS ✯ | Striker 45` and `RS ✯ | Wyvern`

## Description
RS Liveries packages custom aircraft skins created for the RED STAR SQUADRON operations in Digital Combat Simulator (DCS World). These liveries reflect the squadron’s style while making it easy for RS pilots to install and fly with consistent branding.

## How to Download
Visit the [Releases page](../../releases) for the latest downloads. Each release includes an `.exe` installer for easy setup.

## Security
The installer pipeline is fully transparent and open-source (check the .nsi script and GitHub workflows). While we scan for threats, always extract and review files in a safe environment as a precaution.

- Individual livery packages are also provided as `.7z` files in each release, for users who prefer not to download the `.exe` installer.
